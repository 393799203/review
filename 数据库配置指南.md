# PostgreSQL 安装和数据库创建指南

## 🎯 快速开始

PostgreSQL 18 正在后台安装中，请按照以下步骤完成配置：

### 方法一：使用自动脚本（推荐）

```bash
cd /Users/dingyuebo/projects/stock/review
./setup_db.sh
```

这个脚本会自动：
- ✅ 完成PostgreSQL安装
- ✅ 启动PostgreSQL服务
- ✅ 创建stock_review数据库
- ✅ 验证数据库创建成功

### 方法二：手动操作

如果自动脚本遇到问题，可以手动执行：

#### 1. 等待PostgreSQL安装完成

```bash
# 检查安装状态
brew list postgresql@18
```

#### 2. 链接PostgreSQL

```bash
brew link postgresql@18 --force
```

#### 3. 启动PostgreSQL服务

```bash
brew services start postgresql@18
```

#### 4. 创建数据库

```bash
createdb stock_review
```

#### 5. 验证数据库

```bash
psql -l | grep stock_review
```

## 🔧 常见问题

### 问题1：brew link失败

**解决方法**：
```bash
brew unlink postgresql@18
brew link postgresql@18 --force
```

### 问题2：服务无法启动

**解决方法**：
```bash
# 检查服务状态
brew services list

# 重启服务
brew services restart postgresql@18
```

### 问题3：createdb命令失败

**解决方法**：
```bash
# 使用psql创建数据库
psql postgres
CREATE DATABASE stock_review;
\q
```
