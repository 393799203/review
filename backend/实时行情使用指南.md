# 股票实时行情获取功能使用指南

## 功能概述

本项目已成功集成股票实时行情获取功能，支持获取单只股票的实时价格、涨跌幅、成交量等关键数据。

## 实现方案对比

### 方案1：新浪财经API（✅ 推荐，已采用）

**优势：**
- ⚡ **速度极快**：0.07-0.08秒/次
- 🆓 **完全免费**：无需API Key
- 📊 **数据准确**：实时更新
- 🔧 **简单易用**：直接HTTP请求

**数据字段：**
- 股票代码、股票名称
- 最新价、开盘价、最高价、最低价、昨收价
- 涨跌幅、涨跌额
- 成交量、成交额

### 方案2：Akshare全市场接口（❌ 不推荐）

**问题：**
- ⏱️ **速度慢**：60+秒/次（需获取全市场5000+只股票数据）
- 🔄 **效率低**：每次都获取全量数据

### 方案3：同花顺官方API（❌ 不推荐）

**问题：**
- 💰 **收费昂贵**：月费几百～几千元
- 🏢 **门槛高**：主要面向机构用户
- 📝 **申请难**：个人用户几乎无法申请

## API接口文档

### 1. 获取单只股票实时行情

**接口地址：** `GET /api/stock/quote/<stock_code>`

### 方法3：直接调用新浪财经API

```python
import requests
import re

def get_realtime_quote(stock_code):
    # 判断市场
    if stock_code.startswith('6'):
        symbol = f"sh{stock_code}"
    else:
        symbol = f"sz{stock_code}"
    
    # 请求新浪财经API
    url = f"http://hq.sinajs.cn/list={symbol}"
    headers = {
        'Referer': 'http://finance.sina.com.cn',
        'User-Agent': 'Mozilla/5.0'
    }
    
    response = requests.get(url, headers=headers, timeout=5)
    response.encoding = 'gbk'
    
    # 解析数据
    pattern = re.compile(r'="(.*)"')
    match = pattern.search(response.text)
    
    if match:
        data_list = match.group(1).split(',')
        return {
            'code': stock_code,
            'name': data_list[0],
            'price': float(data_list[3]),
            'change_percent': ((float(data_list[3]) - float(data_list[2])) / float(data_list[2]) * 100)
        }
    
    return None

# 使用示例
quote = get_realtime_quote('600519')
print(quote)
```

## 常见问题

### Q1: 数据有延迟吗？
A: 新浪财经API数据实时更新，延迟约3-5秒，适合中低频交易策略。

### Q2: 有调用频率限制吗？
A: 建议控制调用频率，避免过于频繁请求导致IP被封。推荐间隔1秒以上。

### Q3: 支持哪些市场？
A: 支持沪深A股市场（包括主板、创业板、科创板）。

### Q4: 如何获取更多数据字段？
A: 如需Level-2数据、逐笔成交等深度数据，建议：
1. 通过券商量化通道（如QMT、Ptrade）
2. 购买专业数据服务（如Tushare Pro、同花顺iFinD）
