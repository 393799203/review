## 🔧 技术实现

### ArkTS适配

由于ArkTS与React的差异，部分组件做了适配：

| Web组件 | 鸿蒙组件 | 说明 |
|---------|---------|------|
| antd Card | Column + borderRadius | 自定义卡片 |
| antd Table | List/ForEach | 列表渲染 |
| antd Divider | Divider | 分隔线 |
| antd Spin | LoadingProgress | 加载中 |
| antd Refresh | Refresh | 下拉刷新 |
| antd Button | Button | 按钮 |
| antd Input | TextInput | 输入框 |

### 状态管理

- 使用 `@State` 管理组件内部状态
- 使用 `@StorageLink` 实现全局状态（token、user）
- 使用 `AppStorage` 存储应用级数据

### 网络请求

- 统一的 `HttpUtil` 工具类
- `StockApi` 服务类封装所有API调用
- 统一的错误处理和响应处理

## 📐 响应式设计

所有页面都采用了移动端优先的设计：

- 使用百分比布局和flexbox
- 卡片式设计，视觉层次清晰
- 触摸友好的交互区域（至少44px高度）
- 下拉刷新提升用户体验
